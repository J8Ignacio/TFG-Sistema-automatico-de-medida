# TFG-Sistema-automatico-de-medida
En este repositorio se incluye con los archivos necesarios para la ejecución de un proceso de calibración de instrumentos mediante python.
En ella se puede encontrar el archivo local.py y servidor.py, ambos serán los archivos que deberá ejecutar el usuario para iniciar el Sistema de Medida Automático. No obstante se debe de descargar la carpeta entera, puesto que tambien se encuentra los recursos que necesitan la aplicación. La carpeta contiene:
-Local.py: La aplicación para controlar el SAM en local.
-Servidor.py: La aplicación para controlar el SAM en remoto.
-Visa.py: Biblioteca de funcione a las que hace uso local.py y servidor.py.
-IU: Recursos de interfaz gráfica que hace uso la aplicación local.py.
-Templates: Archivos html que necesita la aplicación servidor.py para poder montar la página.
-Static: Recursos gráficos que necesita los archivos html.
